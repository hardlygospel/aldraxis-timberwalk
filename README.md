# 🍄 Aldraxis Timberwalk — Wood Elf Druid, Circle of Spores

> A single-page D&D 5e character reference. Self-contained HTML, no build, no deps.

**Live page:** https://hardlygospel.github.io/aldraxis-timberwalk/

A beautiful one-page reference for a Wood Elf Druid of the Circle of Spores — every
class feature, subclass ability, spell slot, and spell available from level 1 to 20,
laid out in a "haunted forest" dark theme with mushroom-purple accents.

## What's on the page

- **Wood Elf race traits** (Mask of the Wild, Fleet of Foot, Trance, Fey Ancestry…)
- **Druid 1 → 20 progression** — proficiency, features, cantrips known, full spell-slot table
- **Circle of Spores subclass** — Halo of Spores, Symbiotic Entity, Fungal Infestation, Spreading Spores, Fungal Body
- **Always-prepared circle spells** at levels 2 / 3 / 5 / 7 / 9
- **Druid cantrips** — PHB + XGE
- **Every druid spell** from levels 1–9 (PHB, XGE, TCE) with action type, duration, concentration / ritual flags
- **Wild Shape reference** — CR limits and sample forms by druid level
- **Combat quick reference** — opening sequence, reaction economy, necrotic damage stack
- **Quick sheet** — HP formula, AC range, save profs, recommended skills

## Sources

Compiled from:
- *Player's Handbook* (2014)
- *Xanathar's Guide to Everything*
- *Tasha's Cauldron of Everything*

For one-shot reference only — your DM's table is canon.

## Local viewing

```bash
open index.html       # macOS
xdg-open index.html   # Linux
start index.html      # Windows
```

Or just double-click the file. No server required.

## Print

A print stylesheet is included — `Cmd/Ctrl-P` produces a clean character-sheet
companion suitable for the gaming table.
